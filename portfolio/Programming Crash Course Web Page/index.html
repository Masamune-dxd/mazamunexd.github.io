HTML (HyperText Markup Language)

HTML is the standard markup language for documents designed to be displayed in a web browser. It provides the structure for a web page.

<!DOCTYPE html>

Definition: Declares the document type and version of HTML. It must be the first thing in your HTML document.

<!DOCTYPE html>

<html>...</html>

Definition: The root element of an HTML page. All other elements are nested inside this tag.

<html lang="en">
  ...
</html>

<head>...</head>

Definition: A container for metadata about the HTML document, such as the title, character set, and links to stylesheets.

<head>
  <meta charset="UTF-8">
  <title>My Webpage</title>
  <link rel="stylesheet" href="style.css">
</head>

<body>...</body>

Definition: Contains all the visible content of the webpage, such as text, images, links, forms, etc.

<body>
  <h1>Hello, World!</h1>
  <p>This is a paragraph.</p>
</body>

<h1>-<h6>

Definition: Headings tags used to define different levels of headings from most important (h1) to least important (h6).

<h1>Main Title</h1>
<h2>Section Subtitle</h2>

<p>

Definition: A paragraph tag used to group sentences and display text as a block.

<p>This is a simple paragraph of text on a web page.</p>

<a>

Definition: The anchor tag creates a hyperlink to another page or a location within the same page. The `href` attribute specifies the destination URL.

<a href="https://www.example.com">Visit Example.com</a>

<img>

Definition: The image tag is used to embed an image into a web page. The `src` attribute is required and specifies the path to the image file.

<img src="images/logo.png" alt="Company Logo">

<ul>, <ol>, <li>

Definition: `ul` creates an unordered (bulleted) list, `ol` creates an ordered (numbered) list, and `li` defines a list item within either.

<ul>
  <li>Coffee</li>
  <li>Tea</li>
</ul>

CSS (Cascading Style Sheets)

CSS is a stylesheet language used to describe the presentation of a document written in HTML. It controls colors, fonts, layout, and other visual aspects.

color

Definition: Sets the color of the text within an element.

p {
  color: #333333; /* Hex code */
}

background-color

Definition: Sets the background color of an element.

body {
  background-color: #f0f0f0;
}

font-family

Definition: Specifies the font for an element.

h1 {
  font-family: Arial, sans-serif;
}

font-size

Definition: Specifies the size of the font.

p {
  font-size: 16px;
}

margin

Definition: Sets the outer space around an element.

div {
  margin: 10px 20px 30px 40px; /* top right bottom left */
}

padding

Definition: Sets the inner space between an element's content and its border.

.box {
  padding: 15px;
}

border

Definition: Sets the border of an element.

img {
  border: 1px solid black;
}

display

Definition: Specifies the display behavior of an element (e.g., `block`, `inline`, `flex`, `grid`).

.container {
  display: flex;
}

flexbox (flex-direction, justify-content, align-items)

Definition: A one-dimensional layout method for arranging items in rows or columns.

.flex-container {
  display: flex;
  flex-direction: row;
  justify-content: center;
  align-items: center;
}

JavaScript

JavaScript is a programming language that allows you to implement complex features on web pages, such as interactivity and dynamic content.

Variable Declaration (let, const, var)

Definition: Used to declare a variable. `let` is for block-scoped variables, `const` for block-scoped constants, and `var` for function-scoped variables.

let name = "John";
const age = 30;
var city = "New York";

Data Types

Definition: JavaScript has several data types, including `string`, `number`, `boolean`, `array`, and `object`.

let text = "Hello";
let num = 100;
let isTrue = true;
let arr = [1, 2, 3];
let obj = { key: "value" };

Functions

Definition: A block of code designed to perform a particular task. A function is executed when "something" invokes it.

function greet(name) {
  return "Hello, " + name + "!";
}
console.log(greet("Alice"));

If-Else Statements

Definition: A control flow statement that executes a block of code if a specified condition is true.

let x = 10;
if (x > 5) {
  console.log("x is greater than 5");
} else {
  console.log("x is not greater than 5");
}

Loops (for, while)

Definition: Used to execute a block of code repeatedly.

// For loop
for (let i = 0; i < 3; i++) {
  console.log(i);
}

// While loop
let j = 0;
while (j < 3) {
  console.log(j);
  j++;
}

Arrays

Definition: A special type of object used to store multiple values in a single variable. Elements are accessed by an index.

let fruits = ["Apple", "Banana", "Orange"];
console.log(fruits[1]); // Output: "Banana"
fruits.push("Mango");

Java

Java is a high-level, object-oriented programming language. It is known for its "Write Once, Run Anywhere" principle, allowing compiled code to run on any platform that supports Java.

Class

Definition: The fundamental building block of an object-oriented program. It is a blueprint for creating objects.

public class Car {
  String color;
  int year;
}

`main` Method

Definition: The entry point for any Java application. The JVM starts executing from this method.

public static void main(String[] args) {
  System.out.println("Hello, World!");
}

Data Types

Definition: Specifies the size and type of values that can be stored in a variable. Examples include `int` for integers, `double` for decimals, and `String` for text.

int age = 25;
double price = 99.99;
boolean isStudent = true;
String name = "Jane";

Variables

Definition: A container for storing data values. In Java, variables must be declared with their data type.

int speed = 120;
String greeting = "Welcome to Java!";

Methods

Definition: A block of code that performs an action. Methods are defined within a class and are called on an object.

public class Calculator {
  public int add(int a, int b) {
    return a + b;
  }
}

Conditional Statements (if, else if, else)

Definition: Used to execute different code blocks based on a condition.

int time = 20;
if (time < 18) {
  System.out.println("Good day.");
} else {
  System.out.println("Good evening.");
}

Loops (for, while)

Definition: Used to execute a block of code repeatedly.

// For loop
for (int i = 0; i < 5; i++) {
  System.out.println(i);
}

// While loop
int i = 0;
while (i < 5) {
  System.out.println(i);
  i++;
}
